# Ejercicios-Lenguajes-y-compiladores
Ejercicios de la propuesta de ejercicios


# Lenguajes utilizados
- Typescript (Node v22.18)
- C/C++

# Lenguaje TypeScript
### Comando para instalar lo relacionado a typescript y ejecutarlo
```npm install -D typescript ts-node```
### Comando para correr ejercicio numero 1:
```ts-node --compiler-options '{"module":"commonjs"}' exercise-one.ts```
### Comando para correr ejercicio numero 3:
``ts-node exercise-three.ts``

# Lenguaje C/C++
Compilador utilizado:
https://jmeubank.github.io/tdm-gcc/
